# International-Finance-ER-ICA
This repository consists of data and testing for the analysis of any bi/unidirectional relationship evident between inflation and interest rate on the USD/INR interest rate.
The objective was mainly ot check for the existence of PPP and UIP along with some regression models (multivariable OLS, XGBoost etc) to find any other relevant relationships important to our understanding of this project.
The testing concluded with no observed relationship, likely because of being over ridden by other factors such as speculation, oil prices and reserve transaction by the RBI, as well as the tariffs imposed by the US administration.
